  

중복(반복)된 코드를 줄이는데 사용 → 메모리를 덜 차지, 고칠 때 한 부분만 고치면 됨

프로시저, 서브루틴, 함수는 코드를 재사용하는 주요 수단이고, 프로그래밍 내에서는 같은 뜻이다.

한 함수를 여러 번 호출 할 수 있다.

작동 원리

예제를 세제곱 계산하는 함수로 하겠다.

```JavaScript
function
cube(x)
{
	return (x * x * x);
}

y = cube(3);
```
